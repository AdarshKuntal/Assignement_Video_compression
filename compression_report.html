Smart Behavioral Video Compression

Project Overview

  • Input: Classroom CCTV footage (Class_8_cctv_video_1.mov)
  • Duration: Original (120s) → Compressed (9s)
  • Objective: Selective frame retention based on motion and biological presence.

Performance Metrics

Original Size
580 MB
Compressed Size
13 MB
Space Saved
~98%

Technical Implementation

  • Perceptual Hashing (pHash): Eliminates visually redundant frames where no change is detected.
  • Dense Optical Flow (Farneback): Measures pixel-level motion magnitude; filters out static background noise.
  • Haar Cascade Classifiers: Prioritizes human presence by forcing the retention of any frame containing a face.
  • Temporal Context: A baseline frame is captured every 3 seconds to ensure chronological continuity.
  • H.264 Encoding: Final stitching performed via FFmpeg with a 12 FPS target for smooth playback.

Key Observations

  • The algorithm achieved a 98% reduction in file size.
  • Significant storage savings were found during periods of inactivity in the classroom.
  • Human behavioral data (movement and facial recognition) remained intact despite the high compression ratio.
